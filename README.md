# vibestart

**From zero to vibe coding in one paste.**

You just installed VS Code. You want to build something with AI.  
You have never used a terminal. You do not know what npm or git is.  
That is fine.

---

## How it works

1. Install **Kilo Code** in VS Code  
   → [marketplace.visualstudio.com/items?itemName=kilocode.Kilo-Code](https://marketplace.visualstudio.com/items?itemName=kilocode.Kilo-Code)

2. Open Kilo Code chat (`Ctrl+Shift+P` → "Kilo Code: Open Chat")

3. Copy the contents of **[PROMPT.md](PROMPT.md)** and paste it into the chat

4. Answer your agent's questions about your project

5. Type: `Initialize according to custom instructions.`

That is it.

---

## What your agent will set up

- ✅ Checks and guides installation of: git, Node.js, uv
- ✅ Installs [GRACE](https://github.com/osovv/grace-marketplace) — structured AI development methodology
- ✅ Configures [ConPort](https://github.com/GreatScottyMac/context-portal) — persistent memory for your agent
- ✅ Creates `AGENTS.md` — your agent's working instructions
- ✅ Fills in `projectBrief.md` by asking you about your project
